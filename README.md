# Rice Cooker Controller

This project is a Go-based rice cooker controller that allows you to choose different cooking modes and triggers alerts at the end of the cooking process.

## Available Cooking Modes

1. White Rice
2. Brown Rice
3. Steam Cooking
4. Other Food
